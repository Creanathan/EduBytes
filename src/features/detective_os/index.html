Forensic Database Manual

1st Normal Form (1NF) Rules

1. Atomicity: Each cell must contain exactly ONE value. Lists like "Kitchen, Hallway" are illegal. Split them into separate rows.

2. Unique ID: Every row must be uniquely identifiable. Once data is atomic, use Composite Keys (Log_ID + Observation) to secure the registry.

Field Analysis

- Log_ID: The entry timestamp / serial.
- Subject: Currently encrypted [REDACTED].
- Observation: Witness statements. Often contains non-atomic room lists.
